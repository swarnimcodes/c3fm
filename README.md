# c3fm [WIP]

Terminal based file manager in [c3](https://c3-lang.org/)


## Features

- [x] basic directory traversal
- [ ] three pane mode
- [ ] parent directory in left split pane
- [ ] file visualization in right pane
- [ ] vim like search
- [ ] filtering via search
- [ ] file operations:
  - [ ] copy
  - [ ] cut
  - [ ] paste
  - [ ] touch
  - [ ] delete
  - [ ] rename
  - [ ] bulk rename (?)
